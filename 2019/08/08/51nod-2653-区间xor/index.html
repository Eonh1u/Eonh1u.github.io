51nod-2653-区间xor

Posted on 2019-08-08 | In ACM

题目:
给出区间(a,b)，b >= a，求𝑎 𝑥𝑜𝑟 (𝑎+1) 𝑥𝑜𝑟 (𝑎+2)…..𝑥𝑜𝑟 𝑏。
输入
输入2个数：a b，中间用空格分隔(1 <= a <= b <= 10^9)
输出
输出一个答案
输入样例
3 8
输出样例
11

解法：
每个相邻的奇数与偶数xor之后的值是1，所以只要记录下a到b的长度，然后根据a的奇偶进行处理就可以了
