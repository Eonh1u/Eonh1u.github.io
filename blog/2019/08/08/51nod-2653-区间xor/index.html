51nod-2653-区间xor

发表于2019-08-08|更新于2026-03-13
|浏览量:

题目:
给出区间(a,b)，b >= a，求𝑎 𝑥𝑜𝑟 (𝑎+1) 𝑥𝑜𝑟 (𝑎+2)…..𝑥𝑜𝑟 𝑏。
输入
输入2个数：a b，中间用空格分隔(1 <= a <= b <= 10^9)
输出
输出一个答案
输入样例
3 8
输出样例
11

解法：
每个相邻的奇数与偶数xor之后的值是1，所以只要记录下a到b的长度，然后根据a的奇偶进行处理就可以了

#include<stdlib.h>  
#include<cstring>  
using namespace std;  
int f(int l)  
{  
    l/=2;  
    return l % 2;  
}  
int main()  
{  
    int a,b;  
    scanf("%d%d",&a,&b);  
    int l = b - a + 1;  
    //printf("l %d\n", l);  
    int ans;  
    //printf("%d\n",4)  
    if(l & 1)  
    {  
        if(a&1)  
            ans = a ^ (f(l - 1));  
        else  
            ans = b ^ (f(l - 1));  
    }  
    else  
    {  
        if(a&1)  
            ans = a^b^(f(l - 2));  
        else  
            ans = f(l);  
    }  
    printf("%d", ans);  
    return 0;  
}
