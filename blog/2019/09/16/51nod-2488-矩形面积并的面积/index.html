51nod-2488-矩形面积并的面积

发表于2019-09-16|更新于2026-03-13
|浏览量:

题目：
在二维平面上，给定两个矩形，满足矩形的每条边分别和坐标轴平行，求这个两个矩形的并的面积。即它们重叠在一起之后的总的面积。

输入
8个数，分别表示第一个矩形左下角坐标为(A,B)，右上角坐标为(C,D)；第二个矩形左下角坐标为(E,F)，右上角坐标为(G,H)。
保证A<C，B<D，E<G，F<H。
保证所有数的绝对值不超过210^9，矩形并的面积≤210^9。

输出
输出一个数表示矩阵并的面积。
输入样例
-3 0 3 4 0 -1 9 2
输出样例
45
解法：判断长与宽重合边长即可。
