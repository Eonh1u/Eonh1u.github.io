题目：
&emsp;有𝑛个点，第𝑖号点在一维数轴的𝑖坐标位置。你现在在1号点，你需要走𝑘步，移动总长度为𝑆，问这样的方案是否存在。注意：每一步都至少移动1格，且每一步都必须从一个点移动到另一个点上。
<!--more-->
输入
&emsp;第一行三个数n,k,s(2<=n<=10^9,1<=k<=2*10^5,1<=s<=10^18)。
输出
&emsp;如果存在方案，输出YES，否则输出NO。
输入样例
&emsp;10 2 15
输出样例
&emsp;YES
解法：
&emsp;太水了wr>..... 直接贴代码
代码：

```
#include<stdio.h>
using namespace std;
int main()
{
    long long n,k,s;
    scanf("%lld%lld%lld", &n, &k, &s);
    if(n == 1 || s < k)
        {
            printf("NO\n");
            return 0;
        }
    long long sum = k*(n-1);
    if(sum >= s)
        printf("YES\n");
    else
        printf("NO\n");
    return 0;
}
```
